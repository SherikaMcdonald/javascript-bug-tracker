# JavaScript Bug Tracker

## Overview
This project is a simple bug tracking web app built with HTML, CSS, and JavaScript. Users can add bug reports, mark bugs as fixed, and remove bugs from the list. I built this project to show front-end development skills, problem-solving, and clean project structure.

## Features
- Add new bug reports
- Mark bugs as fixed
- Delete bug reports
- View bug status in one place
- Simple and clean interface

## Tools Used
- HTML
- CSS
- JavaScript
- Visual Studio Code
- GitHub

## Why I Built It
I built this project to practice JavaScript logic and create a useful tool that reflects real software workflows. This project also shows my interest in software development, debugging, and technical review.

## What I Learned
- How to connect JavaScript to a web page
- How to update the DOM with user input
- How to manage basic app logic with arrays and objects
- How to organize a small front-end project clearly

## How to Run
1. Clone the repository
2. Open the project folder
3. Open `index.html` in your browser
4. Add bugs and manage them from the page
